# yarnとは


Facebookが作成したパッケージ管理ツールです。npmと互換性があり同じpackage.jsonを使用できます。

# 使い方


### package.jsonの作成


次のコマンドでpackage.jsonを作成します。

```ruby
yarn init
```

### パッケージのインストール


```ruby
yarn
```

こちらのコマンドでpackage.jsonに記述されたパッケージをインストールします。

### パッケージの追加


次のコマンドでpackage.jsonにパッケージをインストールします。

```ruby
yarn add [パッケージ名]
```

パッケージのアンインストール


```ruby
yarn remove
```
